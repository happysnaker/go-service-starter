# Contributing

Thanks for your interest in improving `go-service-starter`.

## Project philosophy

This repository is intentionally:

- small
- readable
- standard-library-first
- production-minded without becoming framework-heavy

Please keep that philosophy in mind when proposing changes.

## Good contributions

- clearer docs and examples
- small operational improvements
- better defaults
- composable middleware examples
- containerization and release ergonomics
- production-hardening guidance

## Changes that need stronger justification

- large dependency additions
- framework-like abstractions
- features that make the starter harder to understand quickly
- repo changes that significantly increase maintenance cost

## Before opening a pull request

- explain the practical problem being solved
- keep scope focused
- update docs when behavior or layout changes

If you are unsure whether an idea fits the repo, open an issue first.
